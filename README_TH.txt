📋 ระบบบันทึกผิดระเบียบ DISCIPLINE PULSE - ฉบับมี Excel Integration
═══════════════════════════════════════════════════════════════════════════════

✨ สิ่งที่เปลี่ยนแปลง:

1️⃣  ปุ่มอัปโหลด Excel
   • เพิ่มแผงด้านซ้าย (Import Panel) สำหรับอัปโหลดไฟล์ Excel
   • รองรับไฟล์ .xlsx และ .xls
   • แสดงจำนวนนักเรียนที่นำเข้าสำเร็จ

2️⃣  ระบบเติมชื่ออัตโนมัติ (Autofill)
   • เมื่อคุณพิมพ์เลขประจำตัว 5 หลัก
   • ชื่อ-สกุลจะเติมอัตโนมัติจากฐานข้อมูล Excel
   • ไม่ต้องพิมพ์ชื่อซ้ำตั้งแต่ต้น

3️⃣  เก็บข้อมูล Excel ใน localStorage
   • ข้อมูลนักเรียนจะเก็บไว้ใน browser storage
   • ใช้ได้ต่อไปแม้ปิดเว็บและกลับมาใหม่
   • เปลี่ยนข้อมูลได้โดยอัปโหลด Excel ใหม่

4️⃣  ไลบรารี XLSX
   • เพิ่มไลบรารี https://cdnjs.cloudflare.com/ajax/libs/xlsx/
   • ใช้อ่านไฟล์ Excel โดยตรงใน browser

═══════════════════════════════════════════════════════════════════════════════

📊 ขั้นตอนการใช้งาน:

1. เปิดเว็บไซต์ Discipline Pulse
2. ด้านซ้าย: กดปุ่ม "📤 เลือกไฟล์ Excel"
3. เลือกไฟล์ "รายชื่อนักเรียน_มัธยมศึกษาตอนปลาย.xlsx"
4. รอให้แสดง "✅ นำเข้าสำเร็จ (18 นักเรียน)"
5. พิมพ์เลขประจำตัว → ชื่อจะเติมอัตโนมัติ
6. เลือกกฎที่ผิด → บันทึก

═══════════════════════════════════════════════════════════════════════════════

🔧 ไฟล์ที่แก้ไข:

📄 index.html
   • เพิ่ม <div class="import-panel"> สำหรับอัปโหลด
   • เปลี่ยน input ชื่อเป็น datalist
   • เพิ่ม <datalist id="studentNameList"> สำหรับ autocomplete
   • เพิ่มโค้ด XLSX library

📄 script.js
   • เพิ่ม STUDENTS_KEY, studentNameList, excelFile
   • เพิ่ม function loadStudents() - โหลดข้อมูลนักเรียน
   • เพิ่ม function updateStudentNameList() - อัปเดต datalist
   • เพิ่ม function handleExcelUpload() - อ่านและเก็บข้อมูล Excel
   • เพิ่ม event listener: studentIdInput blur → autofill ชื่อ
   • เพิ่ม event listener: excelFile change → handle upload

📄 styles.css
   • เพิ่ม .import-panel, .import-description
   • เพิ่ม .file-input-label, .file-button
   • เพิ่ม .import-status, .import-status.is-success/is-error
   • แก้ .workspace grid layout
   • อัปเดต @media query สำหรับ mobile

📊 HOW TO USE:

1. Open Discipline Pulse website
2. Left panel: Click "📤 เลือกไฟล์ Excel"
3. Select "รายชื่อนักเรียน_มัธยมศึกษาตอนปลาย.xlsx"
4. Wait for "✅ นำเข้าสำเร็จ (18 นักเรียน)"
5. Type student ID → Name auto-fills
6. Select rule violation → Submit

⚙️ TECHNICAL DETAILS:

Storage Keys:
  • discipline-pulse-records (violations)
  • discipline-pulse-students (from Excel)
  • discipline-pulse-recorder (staff name)

Excel Structure Expected:
  Column 0: ลำดับ (order)
  Column 1: เลขประจำตัว (student ID)
  Column 2: (empty)
  Column 3: คำนำหน้า (prefix: นาย/นาง/นางสาว)
  Column 4: ชื่อ (first name)
  Column 5: สกุล (last name)
